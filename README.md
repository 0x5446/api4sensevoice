# SenseVoice API Server

这是为SenseVoice编写的API Server，旨在为该项目提供API服务接口。

## 安装

首先，克隆此仓库到你的本地机器：

```bash
git clone https://github.com/你的用户名/你的仓库名.git
cd 你的仓库名
```

然后，使用以下命令安装所需的依赖项：

```bash
pip install -r requirements.txt
```

## 运行

要运行API Server，请使用以下命令：

```bash
python api.py
```
